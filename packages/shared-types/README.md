# @xllmapi/shared-types (types)

Shared TypeScript type definitions — the public contract between all xllmapi packages.

## Modules
- `api/` — HTTP API request/response types
- `protocol/` — WebSocket node protocol messages
- `models/` — Model catalog and definitions

## Usage
```typescript
import type { ChatMessage, CandidateOffering, NodeMessage } from '@xllmapi/shared-types';
```

## Dependencies
None (types only)
