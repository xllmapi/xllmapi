# @xllmapi/web

React frontend — model network, chat, dashboard, admin console.

## Tech Stack
- React 19, React Router 7, Vite, Tailwind CSS 4
- Markdown rendering, Mermaid diagrams

## Pages
- `/` — Landing page
- `/mnetwork` — Model network (platform models + distributed nodes)
- `/chat` — AI chat interface
- `/app` — User dashboard (connection management, node management)
- `/admin` — Admin console

## Communication
Pure HTTP API calls to platform-api. No code dependencies on other packages.
