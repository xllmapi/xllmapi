## 注册与账号

### 如何获取邀请码？

xllmapi 目前采用邀请制注册。你可以通过以下方式获取邀请码：
- 联系已注册的用户索取邀请
- 关注 GitHub 仓库和社区渠道获取公开邀请码
- 参与社区活动获得

### 忘记密码怎么办？

你可以使用邮箱验证码方式登录，无需密码。登录后在个人设置中重新设置密码。

## 模型与调用

### 为什么我连接的模型无法使用？

可能的原因：
- 该模型的所有供应节点都已离线
- 你的 xtokens 余额不足
- API Key 配置有误

请在控制台检查模型状态和账户余额。

### 为什么我供应的模型不显示在网络中？

- 新创建的供应需要管理员审核，请等待审核通过
- 检查供应状态是否为「已启用」
- 如果是分布式节点，确认节点程序正在运行

### 支持哪些模型？

目前网络中包括 DeepSeek、MiniMax、Kimi（Moonshot）等模型。社区可以添加任何兼容 OpenAI 或 Anthropic API 格式的模型。查看 [模型列表](/docs/model-network/models) 获取最新信息。

### 调用返回 404 是什么原因？

404 表示当前没有可用的供应节点提供该模型。可能是：
- 模型名称拼写错误
- 你没有连接该模型
- 所有供应该模型的节点都离线了

### 流式和非流式有什么区别？

- **非流式**：等待模型生成完整回复后一次性返回
- **流式**：模型生成过程中逐 token 返回，用户可以看到实时输出

流式适合交互式场景（如聊天界面），非流式适合批处理场景。

## Token 与余额

### xtokens 是什么？

xtokens 是 xllmapi 平台的内部货币，用于衡量和结算模型调用费用。注册时会获得初始额度。

### 余额不足怎么办？

你可以通过以下方式获取更多 xtokens：
- 供应模型到网络中，每次被调用获得 85% 的费用
- 关注平台活动获取额外奖励

### 为什么同一个模型不同时间的价格不同？

模型调用会被路由到不同的供应节点，不同供应者可能设置了不同的定价。平台会优先选择价格合理且响应快速的节点。

## API 格式

### 应该选择哪种 API 格式？

- 如果你的客户端使用 OpenAI SDK → 选择 **OpenAI 格式**
- 如果你的客户端使用 Anthropic SDK → 选择 **Anthropic 格式**
- 如果你想用一个地址兼容所有格式 → 选择 **xllmapi 统一格式**

### OpenAI 格式和 Anthropic 格式可以混用吗？

可以。你可以在同一个应用中同时使用两种格式，分别调用不同的端点。也可以使用 xllmapi 统一格式，一个 Base URL 自动适配。

### API Key 可以同时用于所有格式吗？

是的。同一个 xllmapi API Key（`xk-xxx`）可以用于所有三种 API 格式，无需分别创建。
