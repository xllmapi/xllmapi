## Bug 报告

如果你在使用过程中遇到问题，请通过 GitHub Issues 提交 Bug 报告：

1. 访问 [github.com/xllmapi/xllmapi/issues](https://github.com/xllmapi/xllmapi/issues)
2. 点击「New issue」
3. 选择「Bug Report」模板
4. 填写以下信息：
   - 问题描述
   - 复现步骤
   - 期望行为
   - 实际行为
   - 环境信息（浏览器、操作系统等）

## 功能建议

如果你有新功能的想法或改进建议：

1. 访问 [github.com/xllmapi/xllmapi/discussions](https://github.com/xllmapi/xllmapi/discussions)
2. 在「Ideas」分类中创建新帖子
3. 描述你的需求和建议方案

社区成员和维护者会参与讨论，优秀的建议会被纳入开发计划。

## 联系方式

- **GitHub Issues**：Bug 报告和技术问题
- **GitHub Discussions**：功能建议和使用讨论
- **QQ 群**：即时交流（群号待公布）

<Callout type="info">
提交反馈时请尽量提供详细信息，这有助于我们更快地定位和解决问题。
</Callout>
