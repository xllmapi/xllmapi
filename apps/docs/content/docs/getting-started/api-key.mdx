---
title: 获取 API Key
description: 创建和管理你的 xllmapi API Key
---

## 创建 API Key

1. 登录后进入控制台
2. 点击左侧菜单「API Key 管理」
3. 点击「创建 API Key」
4. 为你的 Key 起一个名称（如 "我的开发环境"）
5. 点击确认，系统生成 API Key

<Callout type="warn">
API Key 只在创建时完整显示一次，请立即复制并妥善保存。关闭对话框后将无法再次查看完整 Key。
</Callout>

## Key 格式

xllmapi 的 API Key 格式为：

```
xk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

以 `xk-` 开头，后跟一串随机字符。

## 使用方式

API Key 支持两种传递方式：

```bash
# Bearer Token 方式
Authorization: Bearer xk-your-api-key

# x-api-key Header 方式
x-api-key: xk-your-api-key
```

## 删除 API Key

如果你的 Key 泄露或不再需要：

1. 进入「API Key 管理」页面
2. 找到对应的 Key，点击「删除」
3. 确认删除

删除后该 Key 立即失效，使用该 Key 的所有请求将返回 401 错误。
