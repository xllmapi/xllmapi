## 邀请制注册

xllmapi 目前采用**邀请制注册**。你需要一个有效的邀请码才能创建账号。

邀请码可以通过以下方式获取：
- 已有用户邀请
- 社区活动发放
- 关注官方渠道获取公开邀请码

## 注册流程

1. 访问平台首页，点击「注册」
2. 输入邮箱地址和邀请码
3. 系统发送验证码到你的邮箱
4. 输入验证码完成注册

## 登录方式

### 邮箱验证码登录

1. 在登录页输入注册时使用的邮箱
2. 点击「发送验证码」
3. 查收邮件，输入 6 位验证码
4. 登录成功，自动跳转到控制台

### 密码登录

1. 在登录页输入邮箱和密码
2. 点击「登录」

## 首次设置密码

首次注册后，你可以在控制台的个人设置中设置登录密码。设置密码后即可使用密码方式登录，无需每次接收验证码。

<Callout type="info">
即使设置了密码，你仍然可以随时使用邮箱验证码方式登录。
</Callout>
