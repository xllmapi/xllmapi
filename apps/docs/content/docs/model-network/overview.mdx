## 什么是模型网络

模型网络是 xllmapi 的核心机制。它将分散在不同用户手中的 LLM API 资源汇聚成一个统一的模型市场，让每个参与者都能以低成本使用多种模型。

## 两种角色

### 消费者

消费者通过 API 调用模型网络中的模型。每次调用消耗 xtokens（平台内部货币）。

- 注册后自动成为消费者
- 浏览并连接模型网络中的模型
- 通过 API 或内置聊天界面使用模型

### 供应者

供应者向网络提供模型，赚取 xtokens。每个用户都可以同时作为消费者和供应者。

- 将自己的 LLM API Key 接入平台
- 设置模型定价和每日限额
- 获得调用收入的 85%

## 节点类型

模型网络中有两种供应节点：

| | 平台节点 | 分布式节点 |
|---|---|---|
| API Key 存储 | 加密存储在平台 | 保存在本地 |
| 部署方式 | 无需部署 | 需要运行 node-cli |
| 隐私级别 | 标准 | 高（平台无法看到密钥） |
| 可用性 | 平台保障 | 取决于本地运行状态 |
| 适合场景 | 快速接入 | 注重隐私 |

详细说明请参考 [平台节点](/docs/model-network/platform-node) 和 [分布式节点](/docs/model-network/distributed-node)。
