## 什么是平台节点

平台节点是将你的 LLM API Key 托管在 xllmapi 平台上的供应方式。平台使用 AES-256-GCM 加密存储你的密钥，并在用户请求时代为转发。

适合希望快速接入、不想自建服务器的用户。

## 添加供应商凭证

1. 进入控制台，点击「供应管理」
2. 点击「添加供应商」
3. 填写以下信息：
   - **供应商**：选择 LLM 提供商（如 DeepSeek、MiniMax 等）
   - **API Key**：你在供应商处获取的 API Key
   - **Base URL**：供应商的 API 地址（通常自动填充）
4. 点击保存

<Callout type="info">
你的 API Key 使用 AES-256-GCM 加密后存储，平台在转发请求时解密使用。
</Callout>

## 创建模型供应

添加供应商凭证后，创建具体的模型供应：

1. 点击「创建供应」
2. 选择供应商和模型（如 `deepseek-chat`）
3. 设置定价（xtokens / 百万 token）
4. 设置每日调用限额（可选）
5. 提交审核

## 审核流程

新创建的供应需要平台管理员审核：

- 提交后状态为「待审核」
- 管理员会验证 API Key 有效性和模型可用性
- 审核通过后自动上线，状态变为「已启用」
- 如审核不通过，会附带原因说明

## 管理供应

供应上线后，你可以：

- **启用 / 禁用**：临时下线某个供应
- **修改限额**：调整每日调用上限
- **查看收益**：查看该供应的调用统计和 xtokens 收入
- **删除供应**：永久移除该供应
