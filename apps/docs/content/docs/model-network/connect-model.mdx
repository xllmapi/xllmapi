---
title: 连接模型
description: 如何浏览和连接模型网络中的可用模型
---

## 浏览可用模型

1. 登录后进入控制台
2. 点击左侧菜单「模型网络」
3. 页面展示所有可用模型，包括模型名称、供应商、上下文长度和价格

## 连接模型

1. 在模型网络页面找到你想使用的模型
2. 点击模型卡片上的「连接」按钮
3. 确认连接

连接成功后，该模型将出现在：
- 内置**聊天界面**的模型选择列表中
- **API 调用**时可用的模型列表中

## 管理已连接模型

在控制台的「我的模型」或模型网络页面，你可以查看和管理所有已连接的模型：

- 查看每个模型的使用量
- 断开不再需要的模型连接
- 查看模型的定价信息

## 注意事项

<Callout type="info">
连接模型不收费。只有实际发起 API 调用时才会消耗 xtokens。
</Callout>

- 模型的可用性取决于供应节点的运行状态
- 如果某个模型的所有供应节点都离线，调用将返回 404 错误
- 平台会自动在多个供应节点之间进行负载均衡
