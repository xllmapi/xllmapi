## 当前支持的模型

以下是模型网络中当前可用的模型。实际可用性取决于网络中的供应节点。

### DeepSeek

| 模型 | 上下文长度 | 说明 |
|---|---|---|
| `deepseek-chat` | 64K | 通用对话模型 |
| `deepseek-reasoner` | 64K | 推理增强模型 |

### MiniMax

| 模型 | 上下文长度 | 说明 |
|---|---|---|
| `MiniMax-Text-01` | 1M | 百万 token 长上下文模型 |

### Kimi (Moonshot)

| 模型 | 上下文长度 | 说明 |
|---|---|---|
| `moonshot-v1-8k` | 8K | 标准对话模型 |
| `moonshot-v1-32k` | 32K | 长上下文对话模型 |
| `moonshot-v1-128k` | 128K | 超长上下文对话模型 |

## 添加新模型

xllmapi 是一个开放的网络，社区成员可以添加任何兼容 OpenAI 或 Anthropic API 格式的模型。

如果你想添加新的模型供应商：

1. 确保该供应商的 API 兼容 OpenAI 或 Anthropic 格式
2. 通过平台节点或分布式节点接入
3. 创建模型供应并提交审核

<Callout type="info">
模型列表会随着社区成员的贡献不断扩展。你可以在模型网络页面查看最新的可用模型。
</Callout>
