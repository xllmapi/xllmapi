## xtokens

xtokens 是 xllmapi 平台的内部货币，用于衡量和结算模型调用费用。

- 每次 API 调用根据输入和输出的 token 数量消耗 xtokens
- 不同模型的定价不同，由供应者设定
- 可以在控制台查看实时余额

## 初始额度

新用户注册后会获得一定数量的初始 xtokens，用于体验平台功能。你可以使用初始额度进行模型调用，无需立即成为供应者。

## 结算规则

当用户调用某个供应者提供的模型时，消耗的 xtokens 按以下比例分配：

| 分配对象 | 比例 |
|---|---|
| 供应者 | **85%** |
| 平台 | **15%** |

例如：一次调用消耗 100 xtokens，供应者获得 85 xtokens，平台收取 15 xtokens。

## 使用量追踪

在控制台的「用量统计」页面，你可以查看：

- **消费明细**：每次调用的模型、token 数、xtokens 花费
- **供应收入**：作为供应者的调用统计和 xtokens 收入
- **余额变动**：xtokens 的收支记录
- **按日期筛选**：查看特定时间段的使用数据

## 钱包余额

控制台首页展示你的钱包余额：

- **总余额**：当前可用的 xtokens
- **消费总额**：累计消耗的 xtokens
- **供应收入**：累计通过供应模型赚取的 xtokens

<Callout type="info">
当余额不足时，API 调用会返回 402 错误。你可以通过供应模型来赚取更多 xtokens。
</Callout>
