## 三种 API 格式

xllmapi 支持三种 API 格式，你可以根据使用场景选择：

| 格式 | Base URL | 说明 |
|---|---|---|
| OpenAI | `https://api.xllmapi.com/v1` | 兼容 OpenAI Chat Completions API |
| Anthropic | `https://api.xllmapi.com/anthropic/v1` | 兼容 Anthropic Messages API |
| xllmapi 统一 | `https://api.xllmapi.com/xllmapi/v1` | 自动检测格式，一个地址适配所有 |

## 认证方式

所有 API 请求都需要携带 API Key（`xk-xxx` 格式），支持两种方式：

### Bearer Token

```http
Authorization: Bearer xk-your-api-key
```

### x-api-key Header

```http
x-api-key: xk-your-api-key
```

两种方式等效，选择你习惯的即可。

## 速率限制

默认速率限制为 **60 次请求 / 分钟**。超过限制时返回 429 状态码。

响应头中包含速率限制信息：

```http
X-RateLimit-Limit: 60
X-RateLimit-Remaining: 58
X-RateLimit-Reset: 1700000000
```

## 通用响应格式

所有 API 错误返回统一的 JSON 格式：

```json
{
  "error": {
    "message": "错误描述",
    "type": "error_type",
    "code": "error_code"
  }
}
```

详细错误码请参考 [错误码](/docs/api/errors)。
