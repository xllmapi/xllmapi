title: Anthropic 格式
description: 使用 Anthropic Messages 格式调用 xllmapi
---

## 接口地址

```
POST https://api.xllmapi.com/anthropic/v1/messages
```

## 请求头

```http
Content-Type: application/json
x-api-key: xk-your-api-key
anthropic-version: 2023-06-01
```

<Callout type="info">
Anthropic 格式使用 `x-api-key` 头传递 API Key，同时需要 `anthropic-version` 头。
</Callout>

## 请求格式

```json
{
  "model": "MiniMax-Text-01",
  "max_tokens": 1024,
  "messages": [
    {"role": "user", "content": "你好"}
  ]
}
```

### 带 thinking 的请求

部分模型支持 thinking（思考过程）输出：

```json
{
  "model": "MiniMax-Text-01",
  "max_tokens": 16000,
  "thinking": {
    "type": "enabled",
    "budget_tokens": 10000
  },
  "messages": [
    {"role": "user", "content": "分析一下量子计算的前景"}
  ]
}
```

## 响应格式

```json
{
  "id": "msg_xxx",
  "type": "message",
  "role": "assistant",
  "content": [
    {
      "type": "thinking",
      "thinking": "让我分析一下这个问题..."
    },
    {
      "type": "text",
      "text": "量子计算的前景非常广阔..."
    }
  ],
  "model": "MiniMax-Text-01",
  "stop_reason": "end_turn",
  "usage": {
    "input_tokens": 15,
    "output_tokens": 200
  }
}
```

## curl 示例

```bash
curl https://api.xllmapi.com/anthropic/v1/messages \
  -H "Content-Type: application/json" \
  -H "x-api-key: xk-your-api-key" \
  -H "anthropic-version: 2023-06-01" \
  -d '{
    "model": "MiniMax-Text-01",
    "max_tokens": 1024,
    "messages": [
      {"role": "user", "content": "你好，请做一下自我介绍"}
    ]
  }'
```

## 流式输出

设置 `"stream": true`，响应以 SSE 格式返回，事件类型包括：

- `message_start` — 消息开始
- `content_block_start` — 内容块开始
- `content_block_delta` — 内容增量
- `content_block_stop` — 内容块结束
- `message_delta` — 消息状态更新
- `message_stop` — 消息结束
