## 概述

xllmapi 兼容 OpenAI API 格式，因此任何使用 OpenAI SDK 的客户端都可以直接接入，只需修改 `base_url` 和 `api_key`。

## Python (openai 库)

```python
from openai import OpenAI

client = OpenAI(
    api_key="xk-your-api-key",
    base_url="https://api.xllmapi.com/v1"
)

response = client.chat.completions.create(
    model="deepseek-chat",
    messages=[
        {"role": "system", "content": "你是一个有帮助的助手"},
        {"role": "user", "content": "你好"}
    ]
)

print(response.choices[0].message.content)
```

## Node.js (openai 库)

```typescript
import OpenAI from "openai";

const client = new OpenAI({
  apiKey: "xk-your-api-key",
  baseURL: "https://api.xllmapi.com/v1",
});

const response = await client.chat.completions.create({
  model: "deepseek-chat",
  messages: [
    { role: "system", content: "你是一个有帮助的助手" },
    { role: "user", content: "你好" },
  ],
});

console.log(response.choices[0].message.content);
```

## 通用配置要点

无论使用什么客户端，只需要修改以下配置：

| 配置项 | 值 |
|---|---|
| API Base URL | `https://api.xllmapi.com/v1` |
| API Key | `xk-your-api-key` |
| Model | 你已连接的模型名称（如 `deepseek-chat`） |

## 常见客户端

以下客户端均可通过设置 Base URL 接入 xllmapi：

- **ChatGPT-Next-Web** — 设置中填入 API 地址和 Key
- **LobeChat** — 自定义模型提供商配置
- **Chatbox** — API 设置中选择 OpenAI 兼容
- **Continue (VS Code)** — config.json 中配置 OpenAI 提供商
- **Cursor** — 设置中配置自定义 API 端点

具体配置方式请参考各客户端的文档，核心都是设置 Base URL 和 API Key。
