## 环境变量配置

OpenClaw 使用 OpenAI 兼容的环境变量进行配置。

### 设置环境变量

```bash
export OPENAI_API_BASE=https://api.xllmapi.com/v1
export OPENAI_API_KEY=xk-your-api-key
```

### 在 .env 文件中配置

在项目根目录创建或编辑 `.env` 文件：

```bash
OPENAI_API_BASE=https://api.xllmapi.com/v1
OPENAI_API_KEY=xk-your-api-key
```

## 使用

配置完成后，正常启动 OpenClaw 即可。它会自动使用 xllmapi 的端点进行模型调用。

```bash
openclaw
```

## 指定模型

如果需要指定使用的模型，可以在启动时通过参数传入：

```bash
openclaw --model deepseek-chat
```

<Callout type="info">
确保你已经在 xllmapi 模型网络中连接了要使用的模型。
</Callout>
